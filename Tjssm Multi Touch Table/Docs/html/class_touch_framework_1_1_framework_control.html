Detailed Description

Builds up a dictionary of touches and matches them to elements registered.

Each time a touch event happens, call the appropriate function (Added/Updated/...). If you are using a tracking system that returns data as a dictionary, enumerate the dictionary and call the required functions for each update.

PointF  touchPoint,

Updates a touch position. Does not change the object the touch is related to by doing a hittest.

Touch is fixed to one element when added, and should never change. This gives the best responsiveness for resizing and rotating, as the fingers move outside of the element area.

Parameters:

touchPoint The coordinates of the touch in screen space.
